### FilterBadge

```tsx

// Toggle Filter Badge:

<FilterBadge
  type="toggle"
  active={false | true}
  onClick={handleOnclick}
>
  Toggle Filter
</FilterBadge>

// Disclosure Filter Badge:

<FilterBadge
  type="disclosure"
  direction="up | down"
  onClick={handleOnclick}
>
  Toggle Filter
</FilterBadge>

// Add / Remove Filter:

<FilterBadge
  type="add | remove"
  onClick={handleOnclick}
>
  Add / Remove Filter
</FilterBadge>

```

- [x] Meplato dark theme
